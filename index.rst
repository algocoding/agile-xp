익스트림 프로그래밍이란?
========================================

익스트림 프로그래밍(Extreme Programming, **XP** )의 목표는 고객이 원하는 질 좋은 소프트웨어를 시간 안에 개발하는 것이다. 개발자들이 실제로 사용하는 실천 방법들을 결합한 개발 접근법을 제시한다. 

.. note::

    **Extream Programming Explained: Embrace Change** 의 초판에 나오는 **XP** 의 정의

    - **XP** 는 모호한 요구사항이나 빠르게 변하는 요구사항에 직면한 중소 규모의 소프트웨어 개발 팀을 위한 가벼운 방법론이다.
